# Binary Search

This section contains binary search based problems solved in Java with focus on efficient search strategies, boundary handling, and logarithmic optimization techniques.

---

## Topics Covered

* Classic Binary Search
* Search Space Reduction
* Lower Bound / Upper Bound Concepts
* Answer-Based Binary Search
* Edge Case Handling

---

## Problems Included

* Binary Search
* First and Last Occurrence
* Square Root using Binary Search

---

## Approach Focus

Each solution is written with attention to:

* Clean and readable Java implementation
* Correct midpoint calculation
* Proper boundary updates
* Time complexity optimization

---

## Objective

To develop strong understanding of logarithmic problem solving, which is essential for advanced topics such as searching in rotated arrays, optimization problems, and interview-level algorithm design.
