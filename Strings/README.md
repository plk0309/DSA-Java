# Strings

This section contains string-based problems solved in Java with focus on pattern recognition, character-level processing, and efficient manipulation techniques commonly used in interviews.

---

## Topics Covered

* Character Traversal
* Frequency Counting
* String Comparison
* Pattern Matching Basics
* Reversal and Transformation Logic

---

## Problems Included

* Palindrome Check
* Anagram Check
* Reverse String

---

## Approach Focus

Each solution is written with attention to:

* Clean and readable Java implementation
* Efficient use of string operations
* Edge-case handling
* Time and space complexity analysis

---

## Objective

To strengthen understanding of string manipulation techniques, which are essential for problem solving in arrays, hashing, recursion, and advanced algorithmic topics.
